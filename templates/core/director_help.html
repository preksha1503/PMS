How Can We Help?

1. Create Project

Go to Add Project, fill all required details, and click Save Project.

2. View and Filter Projects

Open View Projects and use search/status filter to find records quickly.

3. Assign Project

Open Assign Project, choose a project and employee, then submit assignment.

4. Excel Import

Use Excel Entry to upload `.xlsx` file and import/update project records.
